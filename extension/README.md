# 清洁电器工作台截图助手

MVP 使用方式：

1. 打开 Chrome 或 Edge 的扩展管理页。
2. 开启“开发者模式”。
3. 选择“加载已解压的扩展”。
4. 选择本目录：`extension/`。
5. 打开京东真实商品详情页，选择 SKU 后点击扩展按钮“滚动截图并提交”。
6. 回到工作台，点击“刷新扩展截图”，确认收到截图后再开始分析。

扩展只截取当前浏览器标签页画面，并自动尝试提交到本机 `http://127.0.0.1:4173`、`http://127.0.0.1:4175` 或 `http://127.0.0.1:4174`。
截图会先快速跳到底部探测完整页面高度，再按全页关键位置提交最多 40 张截图；工作台分析时会自动抽样关键截图进入模型，避免把整页切成上百张图片。
